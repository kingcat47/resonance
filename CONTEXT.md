# 공명 (Resonance) - 요양시설 반복학대 익명 신고 시스템

## 핵심 원칙 (절대 위반 금지)

1. 서버는 신고 내용을 절대 평문으로 볼 수 없다. 모든 암호화는 브라우저에서 끝난다.
2. 같은 가해자를 서로 다른 2명 이상이 지목했을 때만("임계값 매칭") 내용이 열린다.
3. 신고자 신원은 신고자 본인이 동의하기 전까지 누구도 볼 수 없다.
4. 지금 단계에서 암호 함수는 전부 더미(자리표시자)다. 실제 암호는 나중 단계에서 넣는다.
   지금은 UI/상태/데이터흐름만 만든다. 임의로 진짜 암호 라이브러리를 넣지 마라.

## 기술 스택 (이미 세팅됨, 바꾸지 마라)

- Vite + React 19 + TypeScript
- SCSS Modules (전역 variables.scss 자동 주입)
- react-router: RootLayout + Outlet 구조 (src/router.tsx)
- 기존 UI 컴포넌트를 반드시 재사용: Button, Input, Checkbox, Typo,
  HStack, VStack, Spacing, Header, MainLayout

## 작업 규칙

- 새 컴포넌트는 기존 폴더 구조와 네이밍 컨벤션을 따른다.
- 기존 UI 컴포넌트로 되는 건 새로 만들지 말고 재사용한다.
- 한 번에 하나의 작업만. 내가 "다음"이라고 하기 전에 다음 단계로 넘어가지 마라.
- 작업 후 무엇을 왜 그렇게 했는지 3줄 이내로 요약해라.

## 시스템 개요

요양시설 내 반복적 학대를 익명으로 신고하는 웹 시스템.
클라이언트 사이드 암호화 + 임계값 매칭으로 신고자 신원과 신고 내용을 보호한다.
